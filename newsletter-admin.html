Add Newsletter via Google Drive

How to add a newsletter from Google Drive

1
Upload your PDF newsletter to Google Drive
2
Right-click the file and select "Get link"
3
Set sharing to "Anyone with the link can view"
4
Copy the link and paste it below
Link Format: Your Google Drive link should look like: https://drive.google.com/file/d/FILE_ID/view
Make sure the link is shared with "Anyone with the link can view"
Optional - will be extracted from the link if left blank
