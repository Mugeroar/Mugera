Course Overview

This 5‑week short course takes you from zero to building your own website. You'll start with computer fundamentals, dive into programming logic, and then focus on web development with HTML, CSS, and JavaScript. By the end you'll have a live project and a certificate (non‑accredited, perfect for skill building).

We emphasise hands‑on practice: you'll write your first "Hello World", use Git/GitHub, work with IDEs, and build a complete website. Ideal for beginners who want to enter tech or add web skills to their toolbox.

What You'll Learn

  • Computer & ICT basics
  • Introduction to programming (logic, variables, functions)
  • First "Hello World" & IDE setup
  • Git, GitHub, version control
  • HTML5, CSS3, responsive design
  • JavaScript fundamentals
  • Final website project

Course Curriculum (5 Weeks)

Week 1: Introduction to Computers & ICT Knowledge

  • Computer hardware & software basics
  • Operating systems (Windows/Linux/macOS)
  • File management and folder structures
  • Internet fundamentals: browsers, URLs, HTTP
  • ICT terminology and digital literacy

Week 2: Intro to Programming & Dev Tools

  • What is programming? Algorithms and logic
  • Setting up an IDE (VS Code)
  • Your first program: "Hello World" in Python/JS
  • Version control with Git & GitHub
  • Basic programming concepts: variables, data types
  • Focus on web development foundations

Week 3: Web Development Core – HTML, CSS, JS

  • HTML5 document structure & semantic tags
  • CSS styling: selectors, box model, flexbox
  • Responsive design with media queries
  • JavaScript basics: DOM manipulation
  • Building a simple interactive page

Week 4: Course Review & Website Project

  • Recap of HTML, CSS, JavaScript
  • Project planning & wireframing
  • Building a multi‑page website
  • Adding interactivity with JS
  • Deploying to GitHub Pages / Netlify

Week 5: Project Review & Certification

  • Present your website project
  • Peer and instructor feedback
  • Final code review & improvements
  • Certificate of completion (short course, non‑accredited)
  • Next steps: advanced JS, frameworks
